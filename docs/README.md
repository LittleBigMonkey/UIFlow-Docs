# UIFlow Documentation Site

This folder contains the public GitHub Pages documentation for UIFlow.

It documents:

- Setup and installation.
- Core concepts.
- Public API reference.
- Samples.
- Transitions.
- Troubleshooting and support.

It intentionally does not publish package source code.

## GitHub Pages Setup

In the GitHub repository settings:

1. Open `Settings > Pages`.
2. Set source to `GitHub Actions`.
3. Push this repository.

The included workflow at `.github/workflows/pages.yml` publishes the `docs/` folder.
