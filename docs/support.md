# Support

Use the UIFlow support forum or issue tracker provided with your package purchase.

When reporting a problem, include:

- Unity version.
- UIFlow version.
- Operating system.
- Whether LitMotion is installed.
- Steps to reproduce.
- Expected result.
- Actual result.
- Console errors.
- Screenshot of the graph when relevant.

## Feature Requests

For feature requests, describe the workflow you are trying to build and why the current API does not cover it. Small examples are more helpful than broad feature names.

## Closed Source Note

Please do not request private implementation source through public support channels. Public documentation covers supported usage, public APIs, and sample patterns.
